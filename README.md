# Library Manager

Simple .NET console application to manage a small library of books - add, list, and search records.

## Features

- Add and manage book records from the command line
- Search, list, and basic persistence using the project's default storage

## Requirements

- .NET 8 SDK (or compatible runtime)

## Build & Run

From the repository root run:

```bash
dotnet build "Library Manager.csproj"
dotnet run --project "Library Manager.csproj"
```

Or from the project folder:

```bash
dotnet run
```

## Files of Interest

- Main entry: [Program.cs](Program.cs)
- Project file: [Library Manager.csproj](Library%20Manager.csproj)

## Development

- Open the project in Visual Studio or VS Code with the C# extension.
- Make changes, then run the build and run commands above to test locally.
